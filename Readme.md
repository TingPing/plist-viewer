PList Viewer
============

Simple application to view property list files.

![plist-viewer screenshot](https://i.imgur.com/rzdC607.png)


Installation
------------

Dependencies:

- Gtk3
- Libplist


```sh
./autogen.sh
make -s
sudo make install
```
